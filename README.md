# CROSSROADS — Gameplay 2.5

Units now use three separate visual representations:

- Far: counter only.
- Medium: stable formation plus one nameplate.
- Close/Inspection: stable formation, nameplate, Pins, and one order chit.

Only one representation is visible at a time. Existing authored formation-slot
coordinates are now enforced inside fixed canvases, preventing responsive CSS
from collapsing squads into vertical lines.

Upload both files to the repository root and wait for `Gameplay 2.5`.
